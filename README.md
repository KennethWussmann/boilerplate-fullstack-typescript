# boilerplate-fullstack-typescript

This is my personal very opionated boilerplate to build a Fullstack TypeScript web app.

Features:
- Clear separation of backend and frontend
- Docker images (Backend, Frontend, ARM64 and AMD64)
- Monorepo (apps, libs)
- GitHub Actions (Lint, Test, Build, Deploy)
- Configuration Management (Env vars, JSON, YAML)
- Logging (JSON structured logs, plain text logs, rotating log files)
- Shadcn/UI
